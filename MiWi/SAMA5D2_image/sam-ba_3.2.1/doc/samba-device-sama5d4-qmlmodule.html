    The SAMBA.Device.SAMA5D4 module contains an implementation of the SAMBA::Device type for the SAMA5D4. device.

    SAMA5D4

    Contains chip-specific information about SAMA5D4 device

    SAMA5D4Config

    Contains configuration values for a SAMA5D4 device

    SAMA5D4EK

    Contains a specialization of the SAMA5D4 QML type for the SAMA5D4x-MB board

    SAMA5D4Xplained

    Contains a specialization of the SAMA5D4 QML type for the SAMA5D4 Xplained Ultra board
