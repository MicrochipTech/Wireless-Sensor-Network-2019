    The SAMBA.Device.SAMA5D3 module contains an implementation of the SAMBA::Device type for the SAMA5D3. device.

    SAMA5D3

    Contains chip-specific information about SAMA5D3 device

    SAMA5D3Config

    Contains configuration values for a SAMA5D3 device

    SAMA5D3EK

    Contains a specialization of the SAMA5D3 QML type for the SAMA5D3x-MB board

    SAMA5D3Xplained

    Contains a specialization of the SAMA5D3 QML type for the SAMA5D3 Xplained board
