    • SAMBA Applet Module

    This module contains applet-specific classes.

    ExtRamConfig

    Contains configuration values for the external RAM applet

    LowlevelConfig

    Contains configuration values for the Low-Level initialization applet

    NANDFlashConfig

    Contains configuration values for the NAND Flash applet

    QSPIFlashConfig

    Contains configuration values for the QSPI Flash applet

    SDMMCConfig

    Contains configuration values for the SD/MMC applet

    SerialConfig

    Contains configuration values for the serial console for all applets

    SerialFlashConfig

    Contains configuration values for the SPI Serial Flash applet
