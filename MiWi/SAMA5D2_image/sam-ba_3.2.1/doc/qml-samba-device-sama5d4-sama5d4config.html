Detailed Description

By default, no configuration values are set.

SD/MMC Configuration

The following SD/MMC configurations are supported:

InstancePeripheralI/O SetBus Width
0HSMCI0 (Slot A)11-bit, 4-bit, 8-bit
0HSMCI0 (Slot B)21-bit, 4-bit
1HSMCI111-bit, 4-bit

The SD/MMC applet on SAMA5D4 does not support voltage switching, so the voltage configuration property is ignored.
